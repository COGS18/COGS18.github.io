## Code Variables != Math Variables

In mathematics, `=` refers to equality (as a statement of truth).

In coding, `=` refers to assignment. 

Math: What is x?

`y = 10x + 2`


Code: What is x?

`x = x + 1`



{:.input_area}
```python


x = 
x = 2
```
