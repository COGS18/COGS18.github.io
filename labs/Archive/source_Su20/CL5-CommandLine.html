CL5: Command Line¶

Welcome to the fifth coding lab!

In this CodingLab we will be working with the command line, and with python files (module files, scripts).

While this may feel like busy work, understanding and completing this coding lab will be essential when it comes time for the final exam and future coding projects you complete. Do your very best to complete all parts of this CodingLab!
